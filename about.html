What is Gob Goblet?

Gob Goblet is a privacy-first, browser-based file encryption tool. It offers seamless drag-and-drop file protection using robust AES-256-GCM and PBKDF2 encryption standards.

The name "Gob Goblet" comes from the biological goblet cell — a mucus-secreting cell that surrounds and protects tissue. In the same way, Gob Goblet surrounds your digital files with an impenetrable protective shell, keeping them safe from prying eyes.

Client-side Encryption

Uses AES-256-GCM directly in your browser. Your files are encrypted locally and never leave your device in plain text.

Server-side Pepper

A secret pepper stored in a Cloudflare Worker hardens the key derivation, defending against offline dictionary attacks.

Open Source

Built with transparency in mind. The frontend and worker source code are openly available for review and contribution.
